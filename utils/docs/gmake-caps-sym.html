NAME

gmake-caps-sym − Make gEDA Capacitor Symbols

SYNOPSIS

gmake-caps-sym [OPTIONS]TEMPLATE[.sym]PARAMETERS[.spm]

DESCRIPTION

gmake-caps-sym Make gEDA Capacitor Symbols

gmake-caps-sym is for ’batch’ production of capacitor symbols for the geda-gaf project. The script requires a template symbol file and a parameter file. Parameters read from the file are used in the output symbol file name or used as replacement values for pre-set variables in the symbol template file.

QUICK

usage: [-h, --help] | [--version] | [options] template[.sym] parameters[.spm] example: gmake-caps-sym electro-example-template.sym params-exported-gnumeric.spm<ENTER>.

PARAMETERS

Fields in the TAB or space delimited parameter file are read and interpreted as follows:

1. Capacitence string to be substituted for ’CAPACITOR_VALUE’ in the symbol file and the output file name 2. Voltage string is included in the file name and as part of the ’CAPACITOR_VALUE’ 3. PartNumber string, included in the file name and in the ’PART_NUMBER’. 4. FootPrint string to be substituted for ’unknown’ in the symbol file

INFORMATION

Use of the following options displays information related to gmake-caps-sym but does not create any symbols:
-h, --help

Show information about the advanced usage of this command.

-u, --usage

Displays examples on usage"

--version

Displays software version information"

DIRECTIVES

-v, --verbose

Include additional information during processing.

OPTIONS

The current version of gmake-caps-sym does not recognize any additional command-line options
