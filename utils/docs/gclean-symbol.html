NAME

gclean-symbol − clean gEDA symbol

SYNOPSIS

gclean-symbol symbol_file

DESCRIPTION

gclean-symbol clean gEDA symbol

usage: gclean-symbol old_symbol_file [ROTATE]> beautyfied_symbol_file
ROTATE will rotate the labels of top and bottom pins

HOMEPAGE

http://www.gpleda.org/
